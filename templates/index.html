Usage de l'api yggnode

Serveur de synchronisation des flux rss et fichiers .torrent pour yggtorrent.li

Le site étant sous protection cloudflare, le projet a pour but de rétablir l'utilisation de ses flux rss :

  • Pour cela il utilise le projet flarrsolverr afin de résoudre le challenge cloudflare et synchroniser régulièrement les flux rss et leurs fichiers .torrent associés sur le serveur.
  • Une api est ensuite mise a disposition pour lire les flux et téléfcharger les .torrent directement depuis ce serveur.

Pour les flux rss

Exemple : {{ server }}/rss?id=2139&passkey=Ttw5xQcCbmjHUwC9jCs8fdbrGEnF8yEt

Deux arguments sont nécessaires dans l'url :

  • ID du flux : Liste des flux synchronisés sur ce serveur
  • Passkey personnel : vous trouverez votre passkey sur votre compte ygg ici, dans la rubrique INFORMATIONS RELATIVES AU TRACKER

Pour faciliter l'usage, vous pouvez utiliser le générateur de lien : LINKS GENERATOR

  • renseignez votre passkey
  • Une fois générés, copiez les liens dans votre client torrent ou lecteur de flux.

Statistiques des dernières mises à jour

Sur la page STATS se trouve, l'heure des dernières synchronisations de flux sur le serveur. De temps à autre cloudflare challenge les flux avec un captcha, il sera donc impossible de faire la synchronisation, c'est en général temporaire. Nous n'avons pour l'instant aucune solution a ce problème.
